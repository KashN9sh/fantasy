## 1. Общая философия боя

Бой в «Тихой тропе» — это **метафора внутреннего столкновения с тревогой**. Здесь нет «победы» в традиционном смысле. Есть:

- **Подавление** — временное затыкание шума, который возвращается сильнее
- **Понимание** — враг становится тише, перестаёт быть врагом
- **Интеграция** — враг превращается в ресурс

Бой не наказывает за проигрыш. Он **показывает последствия** выбора.

## 2. Вход в бой

### Триггеры начала боя

| Тип | Описание | Визуал | Аудио |
|-----|----------|--------|-------|
| **Встреча с врагом** | При активном триггере враг появляется на тропе | Воздух начинает дрожать. Края экрана темнеют. Появляется силуэт врага | Низкий, нарастающий гул. Пульсация |
| **Переход в бой** | Плавное затемнение, переход к карточному столу | Оверворлд размывается. Появляется «стол» — тёмная, но не пугающая поверхность | Гул становится ровным фоном. Исчезают звуки природы |

### Анимация входа

1. **0.0с** — Края экрана начинают темнеть (холодный синий фильтр)
2. **0.5с** — Появляется силуэт врага. Он размыт, не имеет чёткой формы
3. **1.0с** — Оверворлд размывается в шум
4. **1.5с** — Появляется «стол». Карты героя вылетают из колоды и раскладываются веером
5. **2.0с** — Появляется интерфейс: ресурс, рука, колода, сброс. Бой начинается

## 4. Структура хода

### 4.1. Последовательность хода

```
┌─────────────────────────────────────────────────────────────┐
│ 1. НАЧАЛО ХОДА                                              │
│    - Восстановление ресурса (базовое: +1)                   │
│    - Срабатывание эффектов (Усталость, Паника и др.)        │
│    - Срабатывание бафов (Фоновый шум, Ночной разговор и др.)│
│    - Если есть активный Гул — проверка на потерю ресурса    │
├─────────────────────────────────────────────────────────────┤
│ 2. ДОБОР КАРТ                                               │
│    - Добрать до 5 карт (если рука меньше 5)                 │
│    - Если колода пуста — перетасовать сброс                 │
├─────────────────────────────────────────────────────────────┤
│ 3. ДЕЙСТВИЯ ИГРОКА                                          │
│    - Можно сыграть любое количество карт (порядок важен)    │
│    - Каждая карта требует ресурс (если не указано иное)     │
│    - После каждой карты — проверка состояния врага          │
│    - Можно завершить ход в любой момент                     │
├─────────────────────────────────────────────────────────────┤
│ 4. ХОД ВРАГА                                                │
│    - Враг атакует (если не уничтожен/не понят/не сбежал)    │
│    - Применяются эффекты                                     │
│    - Проверка на коалиции (если несколько врагов)           │
├─────────────────────────────────────────────────────────────┤
│ 5. КОНЕЦ ХОДА                                               │
│    - Срабатывание эффектов «до конца хода»                  │
│    - Проверка на победу/поражение                           │
│    - Переход к следующему ходу                              │
└─────────────────────────────────────────────────────────────┘
```

### 4.2. Длительность

- **Стандартный ход:** 30–60 секунд (игрок может думать дольше)
- **Анимации:** Быстрые (0.3–0.5с), не затягивают
- **Ход врага:** 2–5 секунд (враг атакует мгновенно, показывается урон)

## 5. Ресурсная система

### 5.1. Ресурс (Спокойствие/Энергия)

| Параметр | Значение |
|----------|----------|
| **Максимум** | 10 |
| **Начало боя** | 5 |
| **Восстановление за ход** | +1 (базовое). Может быть увеличено бафами |
| **Потеря при атаке врага** | Зависит от врага и карт |
| **Потеря при использовании карт** | 0 (карты не требуют ресурса, если не указано иное) |
| **Критический уровень** | <3 — края экрана краснеют, карты «Пауза» и «Честное я устал» подсвечиваются |

### 5.2. Визуализация ресурса

| Уровень | Цвет | Эффект |
|---------|------|--------|
| 8–10 | Шалфей (#7C9A7A) | Спокойное свечение. Карты в руке имеют чёткие контуры |
| 4–7 | Янтарный (#E0B87A) | Норма. Без изменений |
| 1–3 | Ржавый (#B76E6E) | Края экрана пульсируют красным. Карты слегка дрожат |
| 0 | Тёмно-красный | Поражение |

## 6. Карты в бою

### 6.1. Как играть карты

1. **Выбор:** Нажать на карту в руке. Она подсвечивается
2. **Цель:** Если карта требует цели (враг, союзник) — выбрать цель
3. **Подтверждение:** Карта улетает на стол, срабатывает эффект
4. **Сброс:** Карта уходит в сброс после использования

### 6.2. Типы карт и их поведение

| Тип | Поведение | Анимация | Звук |
|-----|-----------|----------|------|
| **Оружие** | Увеличивает силу следующей атаки/защиты. Действует до конца боя | Карта «экипируется» на стол, появляется иконка оружия рядом с героем | Мягкий звон металла/дерева |
| **Баф (постоянный)** | Добавляет эффект на весь бой | Карта растворяется, иконка бафа появляется в зоне героя | Тёплый, долгий звук |
| **Баф (временный)** | Добавляет эффект на N ходов | Карта растворяется, иконка бафа с таймером | Короткий, нейтральный |
| **Заклинание** | Одноразовый эффект | Карта улетает в цель, визуальный эффект (свет, тени и т.д.) | Магический, «шелестящий» |
| **Защита** | Даёт блок на текущий ход/несколько ходов | Появляется щит вокруг героя | Глухой, твёрдый |
| **Атака** | Наносит урон врагу | Карта летит во врага, враг вздрагивает | Резкий, рубящий |

### 6.3. Комбинации карт

Игрок может сыграть **несколько карт за ход**. Порядок важен:

**Пример:**

1. «Граница» → ставит барьер
2. «Пауза» → усиливает защиту
3. «Список сделанного» → даёт +к атаке за карты в руке
4. «Сжать зубы» → атакует

**Визуальная обратная связь:** После каждой карты — короткая анимация. Карты «складываются» в комбо-индикатор в левом нижнем углу.

## 7. Враг в бою

### 7.1. Отображение врага

| Элемент | Описание |
|---------|----------|
| **Силуэт** | Зависит от врага. Гул — размытые волны. Голос — узнаваемый, но без лица. Сравнение — зеркальная поверхность. Тень — тёмная фигура. Бессонница — круги, петли. Ожидание — множество глаз |
| **Полоска здоровья** | 5–20 единиц, зависит от врага |
| **Уровень** | От 2 до 16. Влияет на силу атак |
| **Эффекты** | Иконки под полоской здоровья (Вина, Стыд, Усталость, Паника) |
| **Состояние** | Активен / Понят (тусклый, не атакует) / Подавлен (мигает, временно исчез) |

### 7.2. Атаки врага

Каждый враг имеет **2–3 типа атак**. Они выбираются случайно, но с весами:

| Тип атаки | Вес | Пример |
|-----------|-----|--------|
| Лёгкая | 50% | 1–2 урона |
| Средняя | 30% | 2–3 урона + эффект |
| Тяжёлая | 20% | 4–5 урона или сильный эффект |

**Анимация атаки:**

1. Враг пульсирует
2. Летит визуальный эффект (волна, тень, взгляд) в сторону героя
3. Герой вздрагивает
4. Полоска ресурса уменьшается
5. Если есть блок — анимация щита, блок уменьшается

### 7.3. Поведение при разных состояниях

| Состояние | Поведение |
|-----------|-----------|
| **Активен** | Атакует каждый ход. Может применять эффекты |
| **Понят** | Не атакует. Может давать советы (текст) или просто присутствовать. Можно выйти из боя без штрафа |
| **Подавлен** | Исчезает на 2–3 хода. Возвращается с +1 уровнем. При возвращении — атакует в тот же ход |
| **Интегрирован** | Врага нет в бою. Карта-интеграция в колоде |

## 8. Несколько врагов в бою

### 8.1. Как это работает

Если триггеры активировали несколько врагов одновременно, они появляются **вместе**.

**Визуал:** Враги располагаются рядом. У каждого своя полоска здоровья. Они могут быть союзниками или врагами друг другу.

### 8.2. Очерёдность ходов

При нескольких врагах:

1. **Ход героя** — стандартный
2. **Ход врагов** — по очереди. Порядок зависит от типа врагов (союзники ходят вместе)

### 8.3. Союзы и вражда

| Тип | Эффект | Визуал |
|-----|--------|--------|
| **Союз** | Враги усиливают друг друга (+2 к силе). Атакуют в одном ходу | Между врагами — связующая линия (цвет: ржавый) |
| **Вражда** | Враги теряют 1 силу в начале хода. Могут атаковать друг друга | Между врагами — рваная, искрящаяся линия |
| **Коалиция** | Три+ врага. Особый эффект (Критик, Петля, Хаос) | Враги сливаются в одну форму, но с несколькими «головами» |

### 8.4. Выход из боя с несколькими врагами

- Можно победить/понять каждого по отдельности
- Если один враг понят — он перестаёт атаковать, но остаётся в сцене
- Если все враги поняты — бой заканчивается автоматически

## 9. Специальные состояния

### 9.1. Эффекты героя

| Эффект | Эффект | Визуал | Снятие |
|--------|--------|--------|--------|
| **Паника** | −2 ко всему, нельзя играть карты «Тишина» | Края экрана дрожат, карты трясутся | Пауза, Дыхание |
| **Вина** | −1 ко всему на 2 хода | Тёмные линии, опутывающие героя | Граница, Разрешение |
| **Стыд** | −2 к защите | Герой «сжимается», становится меньше | Граница |
| **Усталость** | Все карты стоят на 1 дороже | Герой сутулится, движения замедлены | Чай в тишине, Сон |
| **Онемение** | Нельзя играть карты «Просьба о помощи» | Герой бледнеет, контур размыт | Рука на плече |
| **Сожаление** | Карты стоят на 1 дороже | Тень позади героя | Дневник, Спутница |

### 9.2. Эффекты врага

| Эффект | Эффект | Визуал |
|--------|--------|--------|
| **Понят** | Не атакует. Можно выйти из боя | Враг тускнеет, силуэт становится полупрозрачным |
| **Подавлен** | Исчезает на N ходов | Враг разлетается на осколки, но осколки собираются обратно |
| **Усилен** | +1 к уровню (после подавления) | Враг пульсирует красным, становится больше |

## 10. Окончание боя

### 10.1. Условия окончания

| Исход | Условие | Результат |
|-------|---------|-----------|
| **Победа (подавление)** | Все враги побеждены через поглощение | Враги исчезают. Герой получает эффект «Подавление» (следующий бой сложнее). Ресурс восстанавливается на 2 |
| **Победа (понимание)** | Все враги поняты (через принятие) | Враги становятся тише. Герой получает карту интеграции (если враг понят впервые). Ресурс восстанавливается полностью |
| **Победа (уход)** | Использована карта «Прогулка без цели» или «Тишина» | Бой заканчивается без победителя. Враг исчезает (не возвращается) |
| **Поражение** | Ресурс героя = 0 | Герой теряет сознание. Возвращается на последний привал. Теряет 1 карту из колоды (навсегда) |
| **Сбегание** | Сыграна карта «Не сейчас» или выбор в событии | Бой прерывается. Враг остаётся на тропе, возвращается позже с +1 уровнем |

### 10.2. Анимация окончания

**Победа (подавление):**

1. Враг разлетается на осколки
2. Осколки собираются в тёмное облако
3. Облако исчезает
4. Герой стоит, тяжело дышит
5. Появляется текст: *«Ты победил(а). Но враг вернётся.»*

**Победа (понимание):**

1. Враг замирает
2. Силуэт становится мягче, светлеет
3. Враг кивает (или делает жест)
4. Исчезает в свете
5. Появляется текст: *«Ты понял(а). Враг стал тише.»*

**Поражение:**

1. Герой падает на колени
2. Экран темнеет по краям
3. Враги исчезают
4. Герой просыпается на последнем привале
5. Появляется текст: *«Ты потерял(а) сознание. Что-то потеряно навсегда.»*

## 11. Визуальные эффекты боя

### 11.1. Постоянные эффекты

| Эффект | Описание |
|--------|----------|
| **Пульсация краёв** | При низком ресурсе — пульсация ржавым цветом |
| **Дрожание карт** | При эффекте «Паника» карты в руке слегка трясутся |
| **Свечение карт** | Карты, которые можно сыграть с бонусом (например, «Пауза» при низком ресурсе), подсвечиваются |
| **Тени** | У врагов — динамические тени. У Гула — волны. У Тени — расплывчатость |

### 11.2. Анимации карт

| Действие | Анимация |
|----------|----------|
| **Появление карты в руке** | Карта вылетает из колоды, переворачивается, встаёт в руку |
| **Наведение на карту** | Карта поднимается на 10–15 пикселей. Появляется увеличенное описание |
| **Выбор карты** | Карта подсвечивается. Лёгкое свечение по краям |
| **Сыгранная карта** | Карта летит в цель (враг/герой). Оставляет след. Исчезает с эффектом (свет/тень) |
| **Сброс карты** | Карта улетает в стопку сброса с шелестом |

### 11.3. Анимации врага

| Действие | Анимация |
|----------|----------|
| **Появление** | Силуэт собирается из теней/волн/осколков |
| **Атака** | Пульсация → вылет эффекта → попадание |
| **Получение урона** | Враг вздрагивает. Полоска здоровья уменьшается с эффектом «стирания» |
| **Понимание** | Враг светлеет, силуэт становится мягче, полупрозрачным |
| **Подавление** | Враг разлетается на осколки. Осколки собираются в облако |
| **Исчезновение** | Растворяется в свете или тени (в зависимости от типа победы) |

## 12. Аудио-сопровождение

### 12.1. Музыка

| Состояние | Музыка | Описание |
|-----------|--------|----------|
| **Бой (обычный)** | «Тихий гул» | Низкие, ровные ноты. Без резких переходов. Напоминает биение сердца |
| **Бой (несколько врагов)** | «Наслоение» | Те же ноты, но с наложениями. Создаёт ощущение давления |
| **Бой (коалиция)** | «Слияние» | Музыка искажается, появляются диссонансы |
| **Победа (принятие)** | «Облегчение» | Музыка затихает. Появляются высокие, чистые ноты. Слышно дыхание |
| **Победа (поглощение)** | «Тишина» | Музыка резко обрывается. Тишина. Потом — низкий, затяжной звук |
| **Поражение** | «Падение» | Музыка замедляется, искажается, уходит вниз. Тишина |

### 12.2. Звуковые эффекты

| Действие | Звук |
|----------|------|
| **Появление врага** | Низкий, нарастающий гул |
| **Сыгранная карта (принятие)** | Мягкий, «деревянный» звук. Тёплый |
| **Сыгранная карта (поглощение)** | Резкий, «металлический». Холодный |
| **Атака врага** | В зависимости от врага: шёпот, скрежет, удар, шипение |
| **Блок** | Глухой, твёрдый удар. Щит «звенит» |
| **Понимание врага** | Долгий, затихающий звук. Как эхо |
| **Подавление врага** | Резкий, рваный звук. Осколки |
| **Поражение** | Сердцебиение, замедляющееся. Тишина |

## 13. Доступность и обучение

### 13.1. Первый бой (обучение)

- Против Гула без названия
- Пошаговые подсказки: «Сыграй карту», «Заверши ход», «Враг атакует»
- Объясняются базовые механики: ресурс, карты, ход врага
- Объясняется разница между подавлением и пониманием (через выбор)

### 13.2. Подсказки в бою

| Ситуация | Подсказка |
|----------|-----------|
| Низкий ресурс | *«Ты почти без сил. Карты "Пауза" и "Честное я устал" могут помочь.»* |
| Эффект «Вина» | *«Вина ослабляет тебя. "Граница" или "Разрешение" снимут её.»* |
| Враг понят | *«Ты понял(а) этого врага. Он больше не нападает. Можешь выйти из боя.»* |
| Долгий бой | *«Ты держишься. Помни: не обязательно побеждать. Можно уйти.»* |

## 14. Баланс и сложность

### 14.1. Масштабирование

Сложность боя зависит от:

- **Количества активных врагов** (чем больше, тем сложнее)
- **Состояния врагов** (понятые не атакуют, подавленные возвращаются)
- **Сдвига героя** (карты принятия эффективнее против сильных врагов, карты поглощения — против слабых)

### 14.2. «Мягкое» поражение

Поражение не отбрасывает игрока далеко:

- Герой возвращается на последний привал
- Теряет 1 карту из колоды (навсегда)
- Враги, которые были в бою, остаются на тропе (не исчезают)
- Можно попробовать снова с другой стратегией

## 15. Пример боя (сценарий)

**Ситуация:** Герой на тропе. Активны Гул и Голос «ты должен». Гул — фоном, Голос — активен.

**Начало боя:**

> *Воздух дрожит. Ты слышишь знакомый голос: «Ты опять ничего не сделал.»*

**Ход 1:**

- Ресурс: 7/10
- Рука: Пауза, Граница, Дыхание, Сжать зубы, Чай в тишине
- Игрок играет «Границу» → барьер 3
- Игрок играет «Паузу» → +3 к защите, снимает панику (нет)
- Игрок завершает ход
- Голос атакует: *«Ты должен был сделать лучше.»* → 3 урона, но барьер поглощает → ресурс 7
- Гул забирает 1 ресурс (игрок играл карты → нет) → ресурс 7

**Ход 2:**

- Восстановление +1 → ресурс 8
- Добор карт: Список сделанного
- Рука: Дыхание, Сжать зубы, Чай в тишине, Список сделанного + новая
- Игрок играет «Список сделанного» → +4 к атаке (4 карты в руке)
- Игрок играет «Сжать зубы» → +5 к атаке, −1 ресурс → ресурс 7
- Игрок завершает ход
- Голос атакует: *«А что подумают другие?»* → 1 урон (свидетелей нет) → ресурс 6
- Гул забирает 1 ресурс → ресурс 5

**Ход 3:**

- Восстановление +1 → ресурс 6
- Добор: Пауза
- Рука: Дыхание, Чай в тишине, Пауза + 2
- Голос на грани поражения (здоровье 2/10)
- Игрок играет «Дыхание» → +2 ресурса → ресурс 8, снимает панику
- Игрок играет «Паузу» → защита
- Игрок завершает ход
- Голос атакует: *«Ты опять не справился.»* → 2 урона, защита → ресурс 8
- Гул забирает 1 ресурс → ресурс 7

**Ход 4:**

- Восстановление +1 → ресурс 8
- Добор: нет
- Игрок играет «Сжать зубы» → +5, −1 ресурс → ресурс 7
- Голос побеждён (подавлен)
- Бой продолжается с Гулом

**Ход 5:**

- Игрок играет «Чай в тишине» → +3 ресурса → ресурс 10, снимает эффекты
- Игрок завершает ход
- Гул забирает 1 ресурс (игрок играл карты → нет) → ресурс 9

**Ход 6:**

- Игрок играет «Дыхание» → +2 ресурса → ресурс 10
- Игрок завершает ход
- Гул забирает 1 ресурс (игрок играл карты → нет) → ресурс 9

**Игрок понимает, что Гул не победить подавлением.** Меняет стратегию.

**Ход 7:**

- Игрок играет «Паузу» → защита
- Игрок завершает ход
- Гул не забирает ресурс (была сыграна карта)

**Ход 8:**

- Игрок играет «Паузу» → защита
- Игрок завершает ход
- Гул не забирает ресурс

**Ход 9:**

- Игрок играет «Паузу» → защита
- Игрок завершает ход
- Гул не забирает ресурс. Появляется событие: *«Ты слышишь не шум, а ветер. Своё дыхание. Гул рассеивается.»*

**Бой закончен. Гул понят.** Герой получает карту «Фоновый шум».
