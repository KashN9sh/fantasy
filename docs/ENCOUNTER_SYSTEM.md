## 1. Основной принцип

**В оверворлде нет врагов.** Никаких случайных столкновений при шагах.

Враги «нападают» только в одном смысле: они **активируются** как реакция на:

- **Состояние героя** (усталость, бездействие, бегство)
- **Выборы игрока** (ложь, отказ, помощь)
- **Прогресс** (пройденные локации, накопленный сдвиг)
- **Предыдущие бои** (подавление, побег)

**Переход в бой** — это всегда **осмысленное событие**, которое игрок может проследить: «Я соврал Вере → появилась Тень», «Я долго стоял → появился Гул».

**Случайности нет.** Есть **закономерности**, которые игрок может изучить и начать прогнозировать.

---

## 2. Как это выглядит в игре

### 2.1. В оверворлде

Игрок видит:

- Тропу, локации, NPC
- Свой инвентарь, карты
- Индикаторы состояния (ресурс, сдвиг, эффекты)

**Нет никаких вражеских спрайтов.** Никто не ходит по тропе, не ждёт в засаде.

### 2.2. Триггер сработал — переход в бой

**Визуальная и аудио-подготовка:**

1. Появляется **текст-предупреждение** (вверху или в центре экрана)
2. Края экрана начинают темнеть, появляется цветовой фильтр (холодный синий)
3. Возникает **звуковой сигнал** (низкий гул, шёпот, удар сердца — в зависимости от врага)
4. Экран «размывается», оверворлд исчезает
5. Появляется **карточный стол**, карты вылетают из колоды
6. Появляется **силуэт врага** (или нескольких)
7. Начинается бой

**Всё это занимает 2–3 секунды.** Игрок не может «отменить» переход — но он знает, почему он произошёл.

### 2.3. Пример текста перед боем

| Враг | Текст-предупреждение |
|------|---------------------|
| **Гул** | *«Ты стоишь на месте. Воздух начинает дрожать. Гул нарастает.»* |
| **Голос** | *«Знакомый голос: "Ты опять ничего не сделал?"»* |
| **Сравнение** | *«Чьи-то лица мелькают в отражении. Чьи-то успехи.»* |
| **Тень** | *«Тень прошлого поднимается из-за спины. Ты узнаёшь этот момент.»* |
| **Бессонница** | *«Мысли идут по кругу. Ты не можешь остановиться.»* |
| **Ожидание** | *«Тебе кажется, что за тобой следят. Все смотрят.»* |
| **Коалиция** | *«Голоса сливаются в один. Ты окружён(а).»* |

## 3. Полная таблица триггеров «нападения»

### 3.1. Триггеры движения и исследования

| Что делает игрок | Когда срабатывает | Враг | Текст перед боем |
|------------------|-------------------|------|------------------|
| **Долго стоит на месте** | 30 секунд без движения | Гул | *«Ты стоишь на месте. Воздух начинает дрожать. Гул нарастает.»* |
| **Долго идёт без остановки** | 3 локации подряд без паузы | Бессонница | *«Ты идёшь, не останавливаясь. Мысли начинают петлять.»* |
| **Возвращается в посещённую локацию** | Второй раз за игру | Тень | *«Здесь ты уже был(а). Что-то осталось, что ты не доделал(а).»* |
| **Меняет направление 3+ раза** | 3 смены за короткое время | Гул + Бессонница | *«Ты не можешь решиться. Воздух гудит, мысли путаются.»* |
| **Проходит мимо NPC без диалога** | 2 раза подряд | Ожидание | *«Ты прошёл(ла) мимо. Тебе кажется, что они смотрят вслед.»* |

## 4. Как игрок может влиять на частоту боев

| Хочет меньше боев | Хочет больше боев |
|-------------------|-------------------|
| Не стоять на месте | Стоять на месте |
| Отдыхать, когда устал | Игнорировать усталость |
| Говорить правду | Лгать NPC |
| Помогать NPC | Отказывать NPC |
| Понять врага (принятие) | Подавлять врага (поглощение) |
| Использовать карты принятия | Использовать карты поглощения |
| Не менять направление | Ходить туда-обратно |
| Заходить в диалоги со всеми NPC | Проходить мимо NPC |
| Не задерживаться в инвентаре | Долго смотреть инвентарь |

## 5. Частота боёв (примерная)

| Стиль игры | Боёв за игру | Комментарий |
|------------|--------------|-------------|
| **Минималист (осознанное избегание)** | 6–8 | По одному на каждого врага + 1–2 коалиции |
| **Нормальный (естественная игра)** | 10–14 | Некоторые враги возвращаются 1–2 раза |
| **Активный (провокация боёв)** | 16–22 | Враги возвращаются часто, много коалиций |

**Никогда не бывает:** 0 боёв. Даже самый осторожный игрок встретит каждого врага хотя бы раз — через прогресс, сдвиг или неизбежные выборы.

## 6. Почему это лучше, чем случайные столкновения

| Параметр | Случайные столкновения (Final Fantasy) | Система триггеров (Тихая тропа) |
|----------|---------------------------------------|-------------------------------|
| **Связь с сюжетом** | Слабая (гринды) | Прямая (бой — реакция на выбор) |
| **Контроль игрока** | Низкий (нельзя избежать всех) | Высокий (можно влиять через стиль игры) |
| **Метафора тревоги** | Внешняя угроза | Внутреннее состояние |
| **Предсказуемость** | Низкая (случайность) | Высокая (игрок учится триггерам) |
| **Напряжение** | От неожиданности | От последствий выбора |
| **Реиграбельность** | За счёт случайности | За счёт разных стилей игры |

## 7. Итоговая формулировка для документации

> **В «Тихой тропе» нет случайных столкновений.**
>
> В оверворлде отсутствуют вражеские спрайты. Бой — это отдельное состояние, в которое игрок переходит **только при активации триггеров**.
>
> Триггеры привязаны к:
> - **Состоянию героя** (бездействие, усталость, бегство)
> - **Выборам игрока** (ложь, отказ, помощь, молчание)
> - **Прогрессу** (пройденные локации, накопленный сдвиг)
> - **Предыдущим боям** (подавление, побег, поражение)
>
> Каждый бой имеет **текстовое предупреждение**, объясняющее причину.
>
> Игрок может влиять на частоту боёв через стиль игры, но **полностью избежать их невозможно** — тревога всегда найдёт способ проявиться.
