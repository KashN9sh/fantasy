# Визуальные референсы карт «Тихая тропа»

Единый стиль карты: узнаваемый каркас, вариации по пути (**принятие** / **поглощение** / **нейтрально** на базовых). Состав и типы — [DECK.md](./DECK.md). Палитра мира и состояний — [COLOR_SYSTEM.md](./COLOR_SYSTEM.md) (токены: [`tokens.css`](../src/theme/tokens.css), [`canvasPalette.ts`](../src/theme/canvasPalette.ts)). Тон текстов на картах — нейтральный «ты», без навязанного пола ([CONCEPT.md](./CONCEPT.md)).

**Две карты «Тишина»:** визуально развести **№12** (принятие: дымка, силуэт сидящего) и **№31** (редкая: почти белый лист, сидящий со скрещёнными ногами). Разные id в коде — см. [DECK.md](./DECK.md).

---

## Общая стилистика

| Элемент | Описание |
|---------|----------|
| **Формат карты** | Вертикальный прямоугольник, скруглённые углы (радиус 8–10% от ширины) |
| **Фон** | Пиксельная текстура с лёгкой зернистостью. Не однородный — намёк на бумагу / пергамент |
| **Рамка** | Тонкая; цвет от **типа карты** и **пути** (базовые три варианта фона/иконки) |
| **Иконка** | Центральный элемент, пиксель-арт **64×64** или **96×96** px |
| **Название** | Вверху, мелкий, но чёткий шрифт (пиксельный с засечками или моноширинный) |
| **Описание** | Внизу, мелкий текст; курсив для «голоса» карты |
| **Эффект** | Краткая механика в отдельной рамке внизу |
| **Свечение** | Лёгкое по краям при наведении / выборе |
