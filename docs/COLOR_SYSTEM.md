# Цветовая система и состояния

Канонические HEX дублируются в [`src/theme/tokens.css`](../src/theme/tokens.css) и в палитре канваса [`src/theme/canvasPalette.ts`](../src/theme/canvasPalette.ts). При изменении — обновить оба места.

## 1. «На поверхности» (исследование, оверворлд)

Тревога на фоне, не захлёстывает. Тёплая, приглушённая, «утренняя» гамма.

| Элемент            | Цвет                    | HEX       |
| ------------------ | ----------------------- | --------- |
| Фон неба           | Мягкий персиковый       | `#FADDC6` |
| Земля / тропа      | Тёплый серо-коричневый  | `#AA9A8A` |
| Растительность     | Приглушённый оливковый  | `#8A9A6E` |
| Тени / намёк       | Мягкий лавандовый       | `#C9B6D4` |
| Герой / силуэт     | Глубокий индиго         | `#2D3E5A` |

## 2. «Шум» (карточные бои, кризис)

Холодный, сдавленный, «сумерки». Сдвиг в синий и красный.

| Элемент            | Цвет             | HEX       |
| ------------------ | ---------------- | --------- |
| Фон                | Холодный серо-синий | `#4A5B6E` |
| Враги / угрозы     | Ржаво-красный    | `#B76E6E` |
| Эффекты (яд, страх) | Гнилостно-жёлтый | `#C4A45A` |
| Акцент опасности   | Кровяной красный | `#9E4A4A` |
| Контуры            | Холодный графит  | `#3A4A55` |

**Переход:** наложение / смена `data-game-state` (~0.5s), см. `tokens.css`.

## 3. «Тишина» (убежища, отдых, диалоги отшельника)

Тёплый, вечерний, «комнатный».

| Элемент         | Цвет                 | HEX       |
| --------------- | -------------------- | --------- |
| Фон             | Тёплый песочный      | `#EADBC6` |
| Свет            | Мягкий янтарный      | `#E0B87A` |
| Растительность  | Глубокий изумруд     | `#6A8E6A` |
| Вода / ясность  | Спокойный бирюзовый  | `#8AB3B3` |
| Акцент покоя    | Тёплый терракотовый  | `#C68B6E` |

## 5. Финальная сцена (корень тревоги)

Особая гамма; позже можно ветвить `finale-acceptance` | `finale-relief` | `finale-journey`.

| Вариант          | Цвета                         | Смысл        |
| ---------------- | ----------------------------- | ------------ |
| Принятие         | Тёплый серый + мягкий золотой | Тишина, не победа |
| Облегчение       | Бледный голубой + белый       | Пустота без страха |
| Продолжение пути | Глубокий синий + серебряный   | Тревога как попутчик |

Базовая реализация: `data-game-state="finale"` — палитра «принятие» (`#B8B0A8`, `#D4C4A8`).

## Поведение в игре

- Оверворлд → `surface`.
- Вход в карточный бой → `anxiety` (плавный переход).
- Диалог с отшельником, сцена выбора карт (успокоение) → `calm`.
- Экран концовки → `finale`.
- Карты в бою: тип задаётся классами (оружие/баф — шалфей, заклинание — охра, угроза — ржавый) — см. `BattleUI` / YGO-карты.
