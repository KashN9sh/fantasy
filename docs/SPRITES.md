# Таблица спрайтов — «Тихая тропа»

Все визуальные ассеты, необходимые для замены процедурных заглушек на пиксельный арт.
Масштаб игры: 320×180 (×4 zoom), формат — pixel art.

---

## 1. Игрок

Файл в проекте: `Assets/Aseprite/Player.png` — **спрайтшит 192×40 px**, **8 кадров по 24×40**, PPU **16** (кадр в мире 1.5×2.5 юнита). Имена слайсов в Unity: `Player_0` … `Player_7`.

| Кадры | Назначение |
|---|---|
| 0–3 | Ходьба **вправо**; влево — `flipX` |
| 4–7 | **Idle** (анфас, лёгкое покачивание) |

`PlayerController`: поля `walkRightSprites` / `idleSprites` (по 4 ссылки). После смены нарезки в Inspector: меню **Tikhaya Tropa → Player — привязать кадры анимации (Player.png)**.

| Ключ текстуры | Размер (px) | Описание | Анимации | Примечания |
|---|---|---|---|---|
| `player` | см. выше | Персонаж-путник. Мягкий силуэт, без оружия | walk-right 4 кадра, idle 4 кадра, влево — зеркало ходьбы | Заглушка `ph_player`: 12×20 в `GeneratedArt` |

## 4. Фоновые слои (параллакс)

Каждый слой — тайл-спрайт, который повторяется по горизонтали (320×180). Идеально — отдельный набор для каждой локации (9 штук), но можно начать с 3–4 палитровых вариантов.

### Общие слои

| Ключ текстуры | Размер (px) | Описание | scrollFactor | Примечания |
|---|---|---|---|---|
| `bg-sky` | 320×180 | **Небо.** Градиент от тёмно-фиолетового (верх) к индиго (низ). 3–5 «луны»/круга разных размеров и прозрачностей. Мелкие точки-звёзды. | 0 | Статичный. Вариант для каждого palette.bg |
| `bg-mountains` | 320×180 | **Горы.** Силуэты гор/холмов — мягкие треугольные формы, чуть светлее неба. 5–7 вершин, перекрывающих друг друга. | 0.1–0.25 | Медленный параллакс |
| `bg-trees` | 320×180 | **Деревья среднего плана.** Стволы (тонкие вертикальные линии) + округлые кроны. Тёмно-зелёный, с вариациями. | 0.3–0.5 | Средний параллакс |

## 8. Технические требования

- **Формат**: PNG с прозрачностью
- **Стиль**: pixel art, низкое разрешение (см. размеры)
- **Рендер**: `image-rendering: pixelated` (уже в CSS)
- **Палитра**: ограниченная (8–16 цветов на спрайт), тёплые приглушённые тона
- **Настроение**: уютное, мягкое, не пугающее. Без агрессии, без крови, без оружия
- **Анимации**: spritesheet горизонтальный, кадры одного размера
